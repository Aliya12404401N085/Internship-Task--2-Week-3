# Supervised Learning Algorithms Report

## Objective
To understand and implement basic supervised learning algorithms using Scikit-learn.

## Algorithms Implemented
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Dataset
Iris Dataset from Scikit-learn

## Evaluation
The models were trained on the training dataset and evaluated using Accuracy Score.

## Libraries Used
- pandas
- numpy
- scikit-learn
- matplotlib

## Conclusion
This task introduced the fundamentals of supervised learning algorithms and demonstrated how different classification models can be trained and evaluated using Python.
